# 🎓 Education Platform | منصة تعليمية

منصة تعليمية متكاملة تهدف إلى توفير تجربة تعلم آمنة وسلسة للطلاب، مع لوحة تحكم قوية للمعلمين لإدارة المحتوى والاشتراكات.

---

## ✨ المميزات الرئيسية (Main Features)

### 👨‍🎓 للطلاب (For Students):
- **نظام عرض الكورسات**: واجهة عصرية وسهلة التصفح لاستعراض الكورسات المتاحة.
- **مشغل فيديو محمي**: حماية متقدمة ضد تسجيل الشاشة ولقطات الشاشة (Anti-Screen Recording) لضمان حقوق المحتوى.
- **نظام الامتحانات**: امتحانات تفاعلية مع تايمر (Timer) وتصحيح تلقائي للنتائج.
- **تتبع التقدم**: معرفة الدروس المكتملة ونسبة الإنجاز في كل كورس.
- **الشهادات**: إصدار شهادة إتمام كورس تلقائية عند الوصول لنسبة إنجاز 100%.

### 👨‍🏫 للمعلمين/الإدارة (For Admins/Instructors):
- **لوحة تحكم (Dashboard)**: إحصائيات شاملة لعدد الطلاب، الكورسات، والإيرادات.
- **إدارة المحتوى**: إضافة وحذف الكورسات، الدروس، والامتحانات بكل سهولة.
- **إدارة الاشتراكات**: مراجعة طلبات الانضمام للطلاب وقبولها أو رفضها يدوياً.
- **نظام الأسئلة**: بنك أسئلة لكل امتحان مع خيارات متعددة وتحديد الإجابة الصحيحة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

**الواجهة الأمامية (Frontend):**
- React.js (Vite)
- Tailwind CSS (للتصميم العصري)
- React Router (للتنقل بين الصفحات)
- Custom Hooks (لإدارة منطق العمل باحترافية)

**الخلفية (Backend):**
- Node.js & Express
- MySQL (قاعدة البيانات)
- Sequelize ORM (للتعامل مع البيانات)
- JWT (لتأمين الدخول والجلسات)
- Bcrypt (لتشفير كلمات المرور)

## 🚀 التشغيل (Setup & Installation)

### 1. المتطلبات (Prerequisites):
- Node.js مثبت على جهازك.
- MySQL server (مثل XAMPP أو MySQL Workbench).

### 2. إعداد قاعدة البيانات:
- قم بإنشاء قاعدة بيانات باسم `EducationPlatform`.

### 3. إعداد الباك اند (Backend):
```bash
cd BackEnd
npm install
# قم بإنشاء ملف .env بناءً على .env.example واملأ بيانات الـ DB الخاصة بك
npm start
```

### 4. إعداد الفرونت اند (Frontend):
```bash
cd FrontEnd/EducationPlatform
npm install
# قم بضبط VITE_API_URL في ملف .env
npm run dev
```

## 🔒 حماية المحتوى (Security)
تم تطبيق عدة طبقات من الحماية في مشغل الفيديو:
- منع زر Right Click.
- منع الـ Screenshot والـ Screen Recording عبر اكتشاف الـ Visibility API.
- إخفاء المحتوى عند محاولة الطباعة (Print Media).
- منع تشغيل الفيديو في وضع Picture-in-Picture.
